Задача:
Сделать простейший блог. Блог должен состоять из двух страниц: список постов с формой создания и детальный просмотр.
Форма содержит два поля, «Заголовок» и «Текст», оба поля обязательные. После нажатия кнопки «Создать» в форме, запись добавляется в список. 
Если кликнуть на заголовок поста, происходит переход на детальный просмотр. Данные нужно хранить в local storage.
 
Требования:
БЭМ именование классов, React / Redux / webpack / es6+. Красивый код на github
